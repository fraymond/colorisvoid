# Sound Effect Assets

Place your sound effect files here for the CapCut draft generation pipeline.

## Required files

| File | Purpose | Recommended duration |
|------|---------|---------------------|
| `intro.mp3` | Opening jingle played at the start of the video | 2-3 seconds |
| `whoosh.mp3` | Swoosh/transition sound at title card end | < 1 second |
| `title.mp3` | Sound when the title card appears | < 2 seconds |

All files are optional. If a file is missing, the corresponding sound effect is simply skipped.
